Music()
The Music object provides an easy API to play songs from your sounds project directory. By default, the track is looped. 
  Music.play('intro_theme')
static Music.globalVolume(newVolume)
Set the global volume modifier for all music. Any value set is clamped between 0 and 1. This is multiplied into each individual track that plays. If no argument is given return the current global music volume.
Parameters:
newVolume Number Optional
The volume to set
static Music.play(name)
Plays a music track.
Parameters:
name String
The name of the track to play.
static Music.volume(newVolume)
Get or set the current music volume. Any value passed is clamped between 0 and 1. Use this to adjust the volume of individual tracks or to increase or decrease volume during gameplay.
Parameters:
newVolume Number Optional
The volume to set to.
